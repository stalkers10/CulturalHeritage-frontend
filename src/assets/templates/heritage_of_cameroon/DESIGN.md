# Design System Theme

This document outlines the core aesthetic and functional properties of our digital products.

## Color Palette

Our color palette is designed to be elegant and impactful, set in a `dark` mode for a sophisticated user experience.
*   **Primary Color:** `#D4AF37` - A rich, golden hue, signifying importance and interaction.
*   **Secondary Color:** `#8B0000` - A deep, striking red, used for supporting actions and alerts.
*   **Tertiary Color:** `#2E5233` - A calm, earthy green, providing an additional accent.
*   **Neutral Color:** `#12140E` - A deep, almost black tone, forming the foundational background and neutral elements.

## Typography

Our typography system ensures readability and visual hierarchy.
*   **Headlines:** `publicSans` - Modern and highly readable for all headings.
*   **Body Text:** `manrope` - Clear and versatile for general content.
*   **Labels:** `plusJakartaSans` - Crisp and legible for interactive elements and small text.

## Roundedness

UI elements feature a `moderate` level of roundedness (value: `2`), offering a balanced feel that is neither too sharp nor overly soft.

## Spacing

The design system employs a `normal` level of spacing (value: `2`), providing adequate room between elements for clarity without feeling sparse or overly dense. This creates a comfortable and organized visual flow.
